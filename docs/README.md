### 1. Choose Your Documentation Path

**🎯 I want to start using mobile ADP immediately:**
→ Read [Mobile Quick Start Guide](mobile/03-quick-start.md)

**📚 I want to understand the methodology:**

**🔧 I want to see the technical implementation:**
→ Read [Mobile Implementation](mobile/04-implementation.md)

**💡 I want to understand the strategy:**
→ Read [Mobile Strategy](mobile/02-strategy.md)

### 2. Initialize Your Mobile Project

```bash
cd your-mobile-project
adp mobile init
```

ADP will automatically:
- ✅ Detect your mobile platform
- ✅ Set up mobile-specific sensors
- ✅ Configure mobile quality gates
- ✅ Create mobile workflow templates

### 3. Run Mobile Development

```bash
adp mobile run "your feature"
```
